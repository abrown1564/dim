# Manual Skeletonisation Template

Use this file as a reusable frame for transcript-level skeletonisation.

The aim is to capture:

- the live disagreement structure
- the main points of stasis
- the core claims and counterclaims
- the warrants doing argumentative work
- the factual / definitional queue for later verification

This is **not** yet the fallacy layer.

It is the argumentative backbone that later layers should attach to.

## Facts to Verify

Rule:

- each item must be **atomic**
- one factual proposition per bullet
- if a bullet needs `and`, split it into multiple bullets unless separating it would destroy the proposition

Example:

Do **not** write:

- `The Duke of Westminster pays income tax and periodic trust taxes.`

Instead write:

- `The Duke of Westminster pays income tax.`
- `The relevant trust structure is subject to periodic tax charges.`
- `The periodic tax charge is described here as 6% every 10 years.`
